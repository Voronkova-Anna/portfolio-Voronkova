# Проект: Разработка спецификации требований для Chatty 2.0

## Цель проекта:
 -Дополнить и доработать спецификацию требований к веб-приложению Chatty (версия 2.0) с учетом новых функций: 
 -карточки для запоминания слов
 -просмотр видео с примерами произношения слов

## Выполненные задачи: 
-провела анализ требований на основе интервью с заказчиком и пользователями (см. Интервью заметки).
-составила Use Case для функции карточек в графической (диаграмма UML) и табличной формах.
-добавила функциональные требования (FR) к функции карточек с уникальными идентификаторами.
-обновила диаграмму Use Case, включив функцию просмотра видео.
-разработала табличный Use Case для функции просмотра видео с 3 расширениями базового сценария.
-дописала функциональные требования, бизнес-правила и атрибуты качества для новой функции.
-дополнила глоссарий терминами для карточек и видео.
